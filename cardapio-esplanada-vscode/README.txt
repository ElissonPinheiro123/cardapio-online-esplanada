# 🍕 Cardápio Online — Esplanada Restaurante e Pizzaria

Sistema de cardápio online desenvolvido para a Esplanada Restaurante e Pizzaria, permitindo que clientes montem pedidos personalizados e enviem diretamente pelo WhatsApp.

Projeto desenvolvido com foco em experiência do usuário, interface responsiva e fluxo simples de pedido.

## ✨ Funcionalidades

✔ Montagem personalizada de pizzas
✔ Escolha entre tamanho M e G
✔ Seleção de até 2 sabores por pizza
✔ Cobrança automática pelo sabor de maior valor
✔ Carrinho dinâmico
✔ Alteração de quantidade (+ e -)
✔ Remoção de itens
✔ Pesquisa de sabores
✔ Filtro por categorias
✔ Filtro de favoritas
✔ Destaques da casa
✔ Modal rápido para adicionar pizzas
✔ Cálculo automático do pedido
✔ Escolha da forma de pagamento
✔ Troco para dinheiro
✔ Observações gerais do pedido
✔ Integração direta com WhatsApp
✔ Aviso "não fazemos entrega"
✔ Bloqueio fora do horário de funcionamento
✔ Interface responsiva

## 🚀 Como executar

Clone o projeto:

```bash
git clone https://github.com/SEU-USUARIO/NOME-DO-REPOSITORIO.git
```

Entre na pasta:

```bash
cd cardapio-esplanada-vscode
```

Abra no VS Code e depois abra o arquivo `index.html`.

Clique com o botão direito e selecione **Open with Live Server**.

Caso não tenha a extensão, instale **Live Server** no VS Code.

Atendimento: Retirada

1x Pizza G — Meia Calabresa / Meia Filé

Pagamento: PIX

Observação: nenhuma

Total: R$65,00
