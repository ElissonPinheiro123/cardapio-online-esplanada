# Cardápio Online — Esplanada Restaurante e Pizzaria

Projeto de cardápio online para o Esplanada Restaurante e Pizzaria, com interface responsiva, seleção de pizzas, carrinho de pedidos e envio via WhatsApp.

## Autor

- **Elisson Pinheiro**

## Visão geral

Este projeto apresenta um cardápio digital com:

- Montagem personalizada de pedidos
- Escolha de sabores e tamanhos
- Carrinho dinâmico
- Cálculo automático do valor do pedido
- Integração com WhatsApp
- Interface adaptada para dispositivos móveis

## Como executar

1. Clone o repositório:

`ash
git clone https://github.com/ElissonPinheiro123/cardapio-online-esplanada.git
`

2. Entre na pasta do projeto:

`ash
cd cardapio-online-esplanada
`

3. Abra index.html no navegador ou use uma extensão como Live Server no VS Code.
